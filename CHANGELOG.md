## [RC0001] - 2025-12-01

### Added - Core Framework
- **White Paper**: Theoretical foundations and research basis
- **Blue Paper**: Practical applications and implementation guidance
- **Common Speech Intro**: Accessible introduction for general audience
- **Origin Story**: Background on how TLV came to be

### Added - Core Protocols (RC0001)
- **Communication and Speech Protocol**: Framework for understanding personal communication
- **Transcription Protocol**: Methods for capturing spoken content with full dimensionality
- **Translation Protocol**: Guidelines for preserving dimensionality across languages
- **Emotional and Feeling Vocabulary**: Comprehensive terminology for emotional expression
  - Complete A-Z vocabulary entries (26 files)
  - Sample entry template
  - Entry creation template

### Added - Draft Protocols (D0001)
- **Conflict Mediation Protocol**: Specialized application for conflict documentation and resolution
- **Audio Description Protocol**: Accessibility protocol for visual content
- **Captioning Protocol**: Comprehensive captioning methodology

### Added - Examples and Demonstrations
- JFK "We Choose the Moon" speech - TLV transcription (PDF)
- MLK "I Have A Dream" speech - TLV transcription (PDF)
- Nelson Mandela "Rivonia" speech - TLV transcription (PDF)

### Added - Conceptual Documentation
- "Why Three Voices?" - Framework explanation
- "Understanding Observer's Voice: Internal vs External" - Deep dive into witnessing function
- "Why TLV Has No Register" - Design philosophy explanation

### Added - Legal and Licensing
- TLV License Notice (CC BY-NC-SA 4.0)
- Commercial Licensing Information
- Trademark Usage Guidelines

### Documentation Structure
- Organized folder structure: About/, Papers/, Protocols/, Legal/, Licensing/, TLV Vocab/, TLV Transcripts of Speeches/
- Consistent file naming convention with version and language indicators
- Cross-referencing between related documents

### Philosophy and Principles
- Established three foundational axioms
- Defined scope: documentation framework, not therapy
- Emphasized dignity-centered, person-first approach
- Modular protocol architecture for flexible implementation

## Version Naming Convention

- **D####** = Draft (in development, seeking expert review)
- **RC####** = Release Candidate (stable, ready for public use and feedback)
- **V####** = TriLateral Voice (TLV) Foundation Released Version
- Version numbers are sequential starting from 0001
